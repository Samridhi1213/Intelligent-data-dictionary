# Intelligent Data Dictionary Agent

This is a full-stack platform for intelligent database metadata extraction, data quality analysis, and AI-powered documentation.

## Prerequisites
- Python 3.9+
- PostgreSQL
- OpenAI API Key

## Setup Instructions

### 1. Backend Setup
1. Clone the repository.
2. Navigate to the `backend` directory.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Copy `.env.example` to `.env` and fill in your database credentials and OpenAI API Key.
5. Create the database in PostgreSQL (e.g., `olist_db`).

### 2. Dataset Loading
1. Download the [Olist Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) from Kaggle.
2. Extract all CSV files into `backend/data/`.
3. Run the dataset loader:
   ```bash
   python -m backend.services.dataset_loader
   ```

### 3. Run Backend
```bash
python -m backend.main
```
The API will be available at `http://localhost:8000`.

### 4. Frontend Setup
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
The dashboard will be available at `http://localhost:5173`.

## Features
- **Metadata Extraction**: Automatic schema discovery using SQLAlchemy.
- **Data Quality**: In-depth analysis of completeness, uniqueness, and statistics.
- **AI Documentation**: Business-friendly descriptions generated by GPT-3.5.
- **Conversational SQL**: Ask questions in natural language to generate SQL queries.
- **Documentation Export**: Export findings to JSON or formatted Markdown.
